# Dart AI Chat CLI

This is a client-server chat application implemented in **Pure Dart** (standard library only).

## Features
- **TCP Chat Server**: Multiple clients, single room.
- **CLI Client**: ANSI-colored interface with support for commands.
- **AI Personalities**: Autonomous bots powered by Gemini (via raw `HttpClient`).
- **Soft Shutdown**: Graceful exits for AI bots with farewell messages.

## Prerequisites
- Dart SDK installed.
- `GEMINI_API_KEY` set in your environment variables.

## How to Run

1. **Start the Server**:
   ```bash
   dart bin/server.dart
   ```

2. **Start a Client**:
   ```bash
   dart bin/client.dart [YourUsername]
   ```

3. **Interact with AI**:
   Inside the client, use the following commands:
   - `/ai_join critic` (Cynical Carl)
   - `/ai_join polymath` (Professor Spark)
   - `/ai_join poet` (Luna Vane)
   - `/ai_join ?` (Mystery Guest - Dynamically generated by AI)
   - `/exit` (Soft shutdown for yourself)

## Project Structure
- `bin/`: Entry points for server, client, and AI runner.
- `lib/`: Shared logic, AI engine, and configuration.
- `plan/`: Architectural documentation and specifications.
